# My Portfolio Website

[Visit My Portfolio Website]()  

This is my personal portfolio website designed to highlight my software engineering projects, skills, and professional background. The site features interactive sections such as:

- **Main Section**: My profile and a brief introduction.
- **Skills Section**: Displays my technical expertise including Java, JavaScript, React, Node.js, SQL, CSS, HTML, Android Studio, and more.
- **Projects Section**: Showcases my major projects such as the Wedding Website and Android Vacation Planner App.
- **Contact Section**: Includes a functional contact form using the [Resend API](https://resend.com) for email notifications.
- **Responsive Design**: Light/dark mode with animations via [Framer Motion](https://www.framer.com/motion/).

## Tech Stack

- **Frontend:** Next.js (App Router), React, TailwindCSS, TypeScript, Framer Motion  
- **Backend & Integrations:** Node.js, Resend API  
- **Other Tools:** Android Studio, SQL  

## Overview

This portfolio demonstrates:

- Full-stack development skills  
- Front-end design and responsive UI  
- Backend integrations  
- Modern web practices suitable for junior software engineering roles
